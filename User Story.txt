Aufgabe: Klassenhierarchie aufbauen

Klasse Figur
	* Farbe
	* Show()
Status:
	* Erledigt

Klasse Kreis : Figur
	* Radius
Status:
	* Erledigt

Klasse Rechteck : Figur
	* Länge, Breite
Status:
	* Erledigt

Klasse Ring : Kreis
	* innenRadius, aussenRadius
Status:
	* Erledigt

Klasse Parallelogram : Rechteck
	* Länge, Breite, Winkel

Klasse Ellipse : Kreis
	* Radius
	* Stauchungsfaktor
	* Winkel

Anforderung:
	* Die Klassenspezifischen Felder müssen Initialisiert werden (Basisklassen Felder verwenden!)
	* Die Methode Show() muss für jede Klasse die entsprechende Werte ausgeben
	* Ein Array erstellen vom Typ Figur in dem mehrere abgeleitete Figuren von Figur hinzugefügt werden und die Methode Show() ausgeführt wird.
